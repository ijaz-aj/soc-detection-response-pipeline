# SOC Detection & Response Pipeline

An automated detection-and-response pipeline on a fully open-source, self-hosted stack:
OpenSearch SIEM, Sigma detection-as-code, CALDERA emulation, MISP threat intel, Shuffle SOAR,
local LLM triage (Qwen via Ollama), and VECTR purple-team tracking.

> Work in progress - built phase by phase. See PROGRESS.md.
